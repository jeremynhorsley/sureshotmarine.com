What to include (so I can help quickly)

1) Where the boat is

Marina name + slip (or at least the harbor/city). If access requires a gate code or check-in, mention it.

2) The symptom

What’s happening, when it happens, and what changes it (shore power on/off, engine running, loads, weather).

3) Photos

Electrical panel, battery area, affected equipment, and any labels/serial plates. Clear phone photos are perfect.

4) Any recent changes

New batteries, charger, alternator/regulator, solar, inverter, electronics, shore power work, or a yard visit.

What to expect

  • Text is usually the fastest way to start a job and often saves time onsite.
  • I’ll ask a few targeted questions and may suggest quick checks before scheduling a visit.
  • Onboard work is approached methodically: verify, measure, isolate, confirm root cause, then correct cleanly.
  • If a job requires specialized yard services (hauls, through-hulls, major mechanical), I’ll tell you early.

Availability

I schedule work by appointment. Response time varies depending on current jobs and access windows at marinas, but texting a short description with photos is the quickest way to get on the calendar.

If this is a safety issue (burning smell, hot wiring, sparking, repeated breaker trips, or smoke), shut down the affected circuit and disconnect shore power if safe to do so.
